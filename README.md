# Fukurou 🦉

暗号資産（BTC）現物のデイトレードを、LLM に判断させる自律トレーディング bot。
Docker で 24 時間常駐し、まずはペーパートレードで運用する **学習目的（学習7 : 利益3）** の実験プロジェクト。

> ⚠️ 本リポジトリはソフトウェア設計・実装であり、投資助言ではありません。実資金の運用は自己責任で行ってください。

## コンセプト

- **コードは最低限の「安全床」だけを強制し、判断は LLM に広い裁量を与える**
- 売買能力は自作 MCP サーバー（`gmo-coin-mcp`）としてツール提供し、LLM がツール呼び出しで取引する
- LLM は CLI（`claude` / `codex`）シェルアウトで実行（サブスク利用）
- 取引ログは SQLite（機械の真実）＋ Obsidian（人間の知識）の二本立て。振り返りエージェントが知識を育てる

## 安全床（override 不可）

1. 1 トレード最大損失 2%
2. 全ポジションに損切り必須（取引所ネイティブ逆指値を第一）
3. ナンピン禁止
4. 最大ドローダウン -15% で全停止
5. 合計エクスポージャー上限
6. 残高・レート・呼び出し回数上限

## 技術スタック

Kotlin/JVM ・ Ktor ・ Exposed(SQLite) ・ MCP 公式 Kotlin SDK ・ Docker Compose ・ GMO コイン API
